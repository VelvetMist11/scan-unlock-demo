# 3D模型扫码解锁画廊

![项目预览](https://velvetmist11.github.io/scan-unlock-demo/screenshot.png) <!-- 可替换为实际截图链接 -->

一个基于Web的互动式3D模型展示系统，通过扫描二维码解锁不同角色动画。

## ✨ 功能特性

- **移动优先设计**：完美适配手机/平板设备
- **左右滑动切换**：手势操作切换不同场景
- **扫码解锁机制**：使用真实二维码解锁隐藏内容
- **自动保存进度**：本地存储记录解锁状态
- **动态相机适配**：自动调整模型显示尺寸
- **微信兼容**：支持在微信浏览器中运行

## 🚀 快速体验

直接访问在线演示：  
https://velvetmist11.github.io/scan-unlock-demo/

测试二维码：
- `unlock1` - 解锁角色A
- `unlock2` - 解锁角色B 
- `unlock3` - 解锁角色C
